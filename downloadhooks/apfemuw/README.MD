### About
**apfemuw** is a **Freeware** emulator for **APF Electronics Imagination Machine** made by **Enrique Collado**

This emulator runs on **Windows XP,Vista (x86)**

_APF_EMUW emulates in a PC a computer APF Imagination Machine._
